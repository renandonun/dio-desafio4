Introdução ao HTML na prática

< Voltar

HTML significa Linguagem de Marcação de Hipertexto. É uma linguagem de marcação que é usada para estilizar estruturar o conteúdo de uma página da web. HTML não é uma linguagem de programação, mas sim um conjunto de tags que são usadas para indicar ao navegador como exibir o conteúdo. Segundo o Mozilla Developer Network:

"HTML é a base para a maioria dos sites na internet. É uma linguagem simples e fácil de aprender, e é uma habilidade essencial para qualquer pessoa que queira criar ou editar páginas da web." (2023, Introdução ao HTML).

O que são tags?

< Voltar

As tags HTML são usadas para indicar ao navegador como exibir o conteúdo. O nome da tag indica o tipo de conteúdo que está sendo marcado, e o atributo pode ser usado para fornecer informações adicionais sobre o conteúdo.

A tag <font> é uma tag HTML usada para definir a fonte, o tamanho e a cor do texto. Ela foi introduzida no HTML 3.2 e era muito popular na época, mas foi descontinuada no HTML 5. O uso da tag <font> é desaconselhado porque ela é obsoleta e não é mais suportada por todos os navegadores. Além disso, ela pode dificultar a manutenção e a acessibilidade do código HTML. Para definir a fonte, o tamanho e a cor do texto no HTML 5, é recomendável usar as propriedades CSS1 font-family, font-size e color.

Entendendo a comunicação Cliente x Servidor

< Voltar

A comunicação client-server é um modelo de arquitetura de rede versátil e eficiente que é usado em uma ampla variedade de aplicações. Este modelo de arquitetura divide as responsabilidades de processamento entre dois componentes principais: clientes e servidores. O cliente é o componente que solicita os recursos, enquanto o servidor é o componente que fornece os recursos. Os clientes são responsáveis por iniciar as solicitações de serviço e exibir os resultados, enquanto os servidores são responsáveis por fornecer os serviços solicitados e armazenar os dados relacionados.
